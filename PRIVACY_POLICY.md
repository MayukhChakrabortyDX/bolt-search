# Privacy Policy for Bolt Search

Effective date: March 20, 2026

This Privacy Policy explains how Bolt Search handles data when distributed through Microsoft Store.

## Summary

Bolt Search is a local desktop file search app. The app is designed to process search data on your device and does not upload your files to our servers.

## Data We Process

To provide search features, Bolt Search processes file system metadata on your device, including:

- File and folder names
- File and folder paths
- File size
- File date metadata (such as modified and created timestamps)
- File attributes used by filters (such as hidden or read-only)

This processing happens locally on your Windows device.

## Data We Collect

Bolt Search does not collect or transmit personal data to the developer.

Specifically, the app does not:

- Require account creation or sign-in
- Upload file contents
- Upload file names or paths
- Use in-app analytics or advertising SDKs
- Sell personal data

## How Data Is Used

Local file metadata is used only to:

- Run your selected search filters
- Display matching results in the app
- Open selected items in Windows Explorer when requested

## Data Sharing

Bolt Search does not share your file data with third parties.

## Data Retention

Search results and filter state are used by the app for functionality on your device. We do not maintain a remote database of your searches or files.

## Permissions and Access

Bolt Search requires access to locations you choose to search (for example, selected drives or folders). Without file system access, the search feature cannot function.

The app does not access camera, microphone, contacts, or location services.

## Microsoft Store and Platform Data

When you install or use apps from Microsoft Store, Microsoft may collect diagnostic, usage, purchase, or device data under Microsoft policies. That processing is controlled by Microsoft, not by Bolt Search.

See Microsoft Privacy Statement:
https://privacy.microsoft.com/

## Children

Bolt Search is not directed to children under 13.

## Security

Because data processing occurs locally, your files remain on your device unless you choose to share them outside the app.

## Changes to This Policy

We may update this Privacy Policy as the app evolves. The latest version will be made available with the app listing or support materials.
